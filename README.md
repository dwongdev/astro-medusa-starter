## 🚀 How to start?

All commands are run from the root of the project, from a terminal:

| Command                | Action                                     |
| :--------------------- | :----------------------------------------- |
| `cp .env.example .env` | Copy .env example and fill it in           |
| `yarn`                 | Install dependencies                       |
| `yarn dev`             | Start local dev server at `localhost:4321` |
